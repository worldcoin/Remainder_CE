# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities **privately**.

- Email: tfhard@toolsforhumanity.com
- Website: https://hackerone.com/toolsforhumanity

Do **not** open public GitHub issues for security-related reports.

## Scope

The following are considered security issues:
- Soundness (accepting invalid proofs) bugs in the verifier
- Completeness (behavior correctness) bugs in the prover

The following are out of scope:
- Performance issues
- API ergonomics
- Feature requests

## Supported Versions

Only the `main` branch is currently supported.

## Disclosure

We ask that you do not publicly disclose issues until we have had an opportunity to investigate and, if necessary, issue a fix.

## Disclaimer

This project is provided primarily for research and educational purposes. It has not undergone a recent formal security audit of all of its functionality and should not be used in production without independent review.
